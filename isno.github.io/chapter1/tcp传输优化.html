TCP传输优化实践

TCP传输优化实践

在前面章节中分析了TCP连接阶段的优化策略，本节继续讲解TCP 传输阶段优化策略。

纵观整个控制阶段TCP的传输控制阶段，不论慢启动亦或者拥塞避免等，基本运作流程都可以简化为修改最终数据流，控制修改拥塞控制窗口(cwnd)。

通过以上，TCP传输控制的优化方向可以分为以下：调整较为保守的传输控制参数、启动TCP的新特性 如 ECN、升级利用更新的算法如BBRv2
