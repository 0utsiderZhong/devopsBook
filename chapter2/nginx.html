Nginx的HTTP配置详解

Nginx优化配置

升级OpenSSL 1.1+版本

wget https://www.openssl.org/source/openssl-1.1.1q.tar.gz

tar xzvf openssl-1.1.1q.tar.gz
cd openssl-1.1.1q
./config
make && make install

优化openssl路径

./configure \
--prefix=/usr/local/nginx \
--with-http_v2_module \
--with-http_stub_status_module \
--with-http_ssl_module \
--user=www \
--group=www \
--with-openssl=/usr/local/openssl-1.1.1q
