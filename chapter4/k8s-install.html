最佳Kubernetes集群部署实践

最佳Kubernetes集群部署实践

本文中Kubernetes的安装指导，目的为是您了解Kubernetes各个组件交互原理，以便能清晰并解决生产环境中的各类问题，从来开启集群TLS安全认证，并在关键Kubernetes集群服务处采用二进制方式部署。

本次安装实践指导采用Kubernetes最新的v1.26.0版本，

在下面的步骤中，笔者将在三台物理机中部署具有三个节点 最新1.26版本的Kubernetes 集群。

集群的概况：

  • OS：CentOS Linux release 7.9.2009 (Core) 3.10.0-1160.el7.x86_64
  • Kubernetes： v1.26.0
  • Docker： Docker CE 3:20.10.9-3.el7
  • Master节点：192.168.28.208
  • Node节点： 192.168.28.208 92.168.28.136 192.168.28.251

172.20.0.113 这台主机 master 和 node 复用

安装前准备（所有节点）

关闭所有节点的 SELinux

修改 /etc/selinux/config 文件中设置 SELINUX=disabled ，然后重启服务器

开启转发功能

echo 1 > /proc/sys/net/ipv4/ip_forward

确保防火墙关闭

systemctl stop firewalld
systemctl disable firewalld

添加Docker安装源

yum-config-manager --add-repo http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo

安装Docker CE

yum install -y docker-ce
